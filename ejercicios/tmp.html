La imagen que queremos obtener se puede interpretar como cinco filas formadas por diez cruces.

Para generarla se pueden utilizar bucles anidados.

for i in range(5):
    for j in range(10):
        ...
  • El bucle externo (cuya variable de control es i) corresponde a las filas y por eso se ejecuta cinco veces.
  • El bucle interno (cuya variable de control es j) corresponde a los puntos de cada fila y por eso se ejecuta diez veces.

Para dibujar las cruces, habrá que dibujar dos segmentos en cada iteración. Para cada segmento, habrá que conocer sus vértices, que se indican en la imagen siguiente.

Ahora hay que relacionar los valores de i y j con las coordenadas de cada uno de los puntos.

  • Extremo superior izquierda (puntos rojos en el dibujo anterior)
    • Las coordenadas X (0, 60, 120, 180, ...) de los puntos son las mismas para cada fila, es decir que la fórmula sólo depende de j 
      j:     0    1    2    3    4  ...   9     Fórmula
X:     0   60  120  180  240  ...  540 --> 60*j
    • Las coordenadas Y (0, 60, 120, 180, ...) de los centros de los círculos son las mismas para todos los círculos de una fila, es decir que la fórmula sólo depende de i 
      i:     0    1    2    3    4     Fórmula
Y:     0   60  120  180  240 --> 60*i
  • Extremo inferior derecha (puntos azules en el dibujo anterior)
    • Las coordenadas X (40, 100, 160, 220, ...) de los puntos son las mismas para cada fila, es decir que la fórmula sólo depende de j 
      j:     0    1    2    3    4  ...   9     Fórmula
X:    40  100  160  220  280  ... 580 --> 60*j+40
    • Las coordenadas Y (40, 100, 160, 220, ...) de los centros de los círculos son las mismas para todos los círculos de una fila, es decir que la fórmula sólo depende de i 
      i:     0    1    2    3    4     Fórmula
Y:     40 100  160  220  280 --> 60*i+40

    • Las coordenadas X (40, 100, 160, 220, ...) de los puntos son las mismas para cada fila, es decir que la fórmula sólo depende de j

    • Las coordenadas Y (0, 60, 120, 180, ...) de los centros de los círculos son las mismas para todos los círculos de una fila, es decir que la fórmula sólo depende de i

Finalmente, el bucle anidado podría ser:

for i in range(5):
    for j in range(10):
        document["imagen"] <= svg.line(x1=60*j, y1=60*i, x2=60*j+40, y2=60*i+40,
                                      stroke="black", stroke_width=1)
        document["imagen"] <= svg.line(x1=60*j+40, y1=60*i, x2=60*j, y2=60*i+40,
                                      stroke="black", stroke_width=1)
