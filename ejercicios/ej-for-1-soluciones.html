A. Bucles

Bucle for (1) - A-1

print("PARES E IMPARES")
numero_1 = int(input("Escriba un número entero: "))
numero_2 = int(input(f"Escriba un número entero mayor o igual que {numero_1}: "))

if numero_2 < numero_1:
    print(f"¡Le he pedido un número entero mayor o igual que {numero_1}!")
else:
    for i in range(numero_1, numero_2 + 1):
        if i % 2 == 0:
            print(f"El número {i} es par")
        else:
            print(f"El número {i} es impar")

Comentarios

  • La estructura general de este programa es similar a la del resto de programas de la página: se pide un número (o dos) y se hace una comprobación inicial de ese valor (o valores), para mostrar un mensaje de error o ejecutar el núcleo del programa. En este caso, la comprobación es que el segundo número sea mayor que el primero. 
    print("PARES E IMPARES")
numero_1 = int(input("Escriba un número entero: "))
numero_2 = int(input(f"Escriba un número entero mayor o igual que {numero_1}: "))

if numero_2 < numero_1:
    print(f"¡Le he pedido un número entero mayor o igual que {numero_1}!")
else:
    ...
  • Una vez comprobado que los valores son correctos, el núcleo del programa realiza un bucle desde el primer número introducido al segundo. 
        for i in range(numero_1, numero_2 + 1):
        ...
  • En el cuerpo del bucle, se comprueba en cada iteración si la variable de control i es par o no (comprobando si el resto de la división entre 2 es 0 o no) y se imprime el mensaje correspondiente. 
            if i % 2 == 0:
            print(f"El número {i} es par")
        else:
            print(f"El número {i} es impar")

Bucle for (1) - A-2

print("DIVISORES")
numero = int(input("Escriba un número entero mayor que cero: "))

if numero <= 0:
    print("¡Le he pedido un número entero mayor que cero!")
else:
    print(f"Los divisores de {numero} son ", end="")
    for i in range(1, numero + 1):
        if numero % i == 0:
            print(i, end=" ")

Comentarios

  • Para encontrar los divisores del número indicado por el usuario, el programa comprueba para cada número entre 1 y el número si el resto de la división es cero. 
        for i in range(1, numero + 1):
        if numero % i == 0:
            ...
  • Si lo es, lo escribe: 
            if numero % i == 0:
            print(i, end=" ")
  • Como los números se van escribiendo en la ejecución del bucle, el comienzo de la respuesta se escribe antes del bucle: 
        print(f"Los divisores de {numero} son ", end="")

Programa mejorado

Como los divisiores de un números son siempre inferiores o igual a la mitad del número (exceptuando el propio número, claro), no hace falta comprobar todos los valores desde 1 hasta el número, basta con probar hasta la mitad del número.

print("DIVISORES")
numero = int(input("Escriba un número entero mayor que cero: "))

if numero <= 0:
    print("¡Le he pedido un número entero mayor que cero!")
else:
    print(f"Los divisores de {numero} son ", end="")
    for i in range(1, round(numero/2)+1):
        if numero % i == 0:
            print(i, end=" ")
    print(numero)

Comentarios

  • Como el tipo range() requiere números enteros, para indicar el límite superior del range() (la mitad del número indicado por el usuario) se debe utilizar la función round(), que redondea al entero más próximo (no exactamente, como se comenta en la lección de operaciones aritméticas, ya que los números XXX,5 se redondean al entero par más próximo, pero en este caso eso no provoca problemas). 
        for i in range(1, round(numero/2)+1):
  • Como el número es divisor de sí mismo, una vez completado el bucle, el programa escribe el número. 
        print(numero)

Bucle for (1) - A-3

print("MAYORES QUE EL PRIMERO")
valores = int(input("¿Cuántos valores va a introducir? "))
if valores < 1:
    print("¡Imposible!")
else:
    primero = int(input("Escriba un número: "))
    for i in range(valores - 1):
        numero = int(input(f"Escriba un número más grande que {primero}: "))
        if numero <= primero:
            print(f"¡{numero} no es mayor que {primero}!")
    print("Gracias por su colaboración")

Comentarios

  • La petición del primer número se debe hacer antes del bucle puesto que sólo se pide una vez. 
            numero = int(input(f"Escriba un número más grande que {primero}: "))
  • El bucle debe repetirse una vez menos que el número de valores a introducir (ya que uno de los valores, el primero, se ha introducido antes del bucle). 
        for i in range(valores - 1):

Bucle for (1) - A-4

print("MAYORES QUE EL ANTERIOR")
valores = int(input("¿Cuántos valores va a introducir? "))
if valores < 1:
    print("¡Imposible!")
else:
    primero = int(input("Escriba un número: "))
    for i in range(valores - 1):
        numero = int(input(f"Escriba un número más grande que {primero}: "))
        if numero <= primero:
            print(f"¡{numero} no es mayor que {primero}!")
        primero = numero
    print("Gracias por su colaboración")

Comentarios

  • La única diferencia (aparte del título) de este programa con el anterior es que en este no se compara con el primer número, sino con el último que se ha escrito. Para conseguirlo, después de hacer la comparación, el programa guarda en la variable primero el valor que acaba de escribir el usuario. Así, en la siguiente iteración, al hacer la comparación se utilizará el último valor escrito. 
            primero = numero

B. Contadores

Bucle for (1) - B-1

print("NÚMEROS NEGATIVOS")
numero = int(input("¿Cuántos valores va a introducir? "))

if numero <= 0:
    print("¡Imposible!")
else:
    contador = 0
    for i in range(1, numero + 1):
        valor = float(input(f"Escriba el número {i}: "))
        if valor < 0:
            contador = contador + 1
    if contador == 0:
        print("No ha escrito ningún número negativo")
    elif contador == 1:
        print("Ha escrito 1 número negativo")
    else:
        print(f"Ha escrito {contador} números negativos")

Comentarios

  • El núcleo del programa utiliza un bucle para pedir al usuario el número de valores indicado por el usuario 
        for i in range(1, numero + 1):
        ...
  • Al pedir el número que se va a sumar, se utiliza la variable de control i: 
            valor = float(input(f"Escriba el número {i}: "))
  • El programa utiliza un contador ( contador) para llevar la cuenta de los números negativos introducidos por el usuario.
  • Antes de pedir los valores el contador contador se pone a 0. 
        contador = 0
  • En cada iteración, se comprueba si el valor introducido es negativo. 
            if valor < 0:
            ...
  • Si lo es, se aumenta el contador en 1. 
                contador = contador + 1
  • Una vez completado el bucle, se escribe el resultado. Como la frase que se escribe depende de si se han escrito 0, 1 o 2 números negativos, el programa utiliza una estructura if ... elif ... else ... 
        if contador == 0:
        print("No ha escrito ningún número negativo")
    elif contador == 1:
        print("Ha escrito 1 número negativo")
    else:
        print(f"Ha escrito {contador} números negativos")

Bucle for (1) - B-2

print("CONTADOR DE PARES E IMPARES")
valores = int(input("¿Cuántos valores va a introducir? "))
if valores < 1:
    print("¡Imposible!")
else:
    pares = 0
    for i in range(1, valores + 1):
        numero = int(input(f"Escriba el valor {i}: "))
        if numero % 2 == 0:
            pares += 1
    print(f"Ha escrito {pares} números pares y {valores - pares} números impares")
    print("Gracias por su colaboración")

Comentarios

  • El contador de números pares se debe inicializar antes del bucle. 
        pares = 0
  • El contador de números pares debe aumentar cuando el número escrito es par. 
            if numero % 2 == 0:
            pares += 1
  • No es necesario crear un contador de números impares, ya que los números que no son pares serán impares. 
        print(f"Ha escrito {pares} números pares y {valores - pares} números impares")

Bucle for (1) - B-3

print("NÚMERO PRIMO")
numero = int(input("Escriba un número entero mayor que 1: "))

if numero <= 1:
    print("¡Le he pedido un número entero mayor que 1!")
else:
    contador = 0
    for i in range(1, numero + 1):
        if numero % i == 0:
            contador = contador + 1
    if contador == 2:
        print(f"{numero} es primo")
    else:
        print(f"{numero} no es primo")

Comentarios

  • El algoritmo del programa se basa en que un número primo sólo tiene dos divisores: el 1 y el propio número. El programa cuenta los divisores del número y si sólo hay dos, es que el número es primo
  • El número de divisores se obtiene mediante un bucle desde 1 hasta el número y un contador contador que aumenta en 1 cada vez que el resto de la división del número por la variable de control i da 0. 
        contador = 0
    for i in range(1, numero + 1):
        if numero % i == 0:
            contador = contador + 1
  • Una vez completado el bucle, una estructura if ... else ... escribe si el número es primo o no. 
        if contador == 2:
        print(f"{numero} es primo")
    else:
        print(f"{numero} no es primo")

Programa mejorado

Para que el programa vaya mucho más rápido, se podría probar únicamente la divisibilidad por los números hasta la raíz cuadrada del número, puesto que si es divisible por algún número mayor que su raíz cuadrada, lo será también por un número menor que su raíz cuadrada.

print("NÚMERO PRIMO")
numero = int(input("Escriba un número entero mayor que 1: "))

if numero <= 1:
    print("¡Le he pedido un número entero mayor que 1!")
else:
    contador = 0
    limite = round(numero ** 0.5)
    for i in range(1, limite + 1):
        if numero % i == 0:
            contador = contador + 1
    if contador == 1:
        print(f"{numero} es primo")
    else:
        print(f"{numero} no es primo")
  • Como el tipo range() requiere números enteros, se debe utilizar la función round(), que redondea al entero más próximo. Para calcular la raíz cuadrada, se puede elevar el número a la potencia 0.5.
  • En este caso, será primo si el contador vale al final 1, ya que al probar sólo hasta la raíz cuadrada del número, el bucle no prueba el propio número (que es el segundo divisor de un primo).
  • La diferencia en velocidad es más evidente con números grandes. Por ejemplo, con el número 122949823, mi ordenador tarda 35 segundos en el primer programa y 7 milésimas con el segundo programa.
  • El programa podría ser más rápido todavía (detectando los números que no son primos) si detuviera las comprobaciones en cuanto encontrara un divisor, sin necesidad de probar todos los números hasta la raíz cuadrada, pero para eso hace falta la estructura de control while, que se verá más adelante.

C. Acumuladores

Bucle for (1) - C-1

print("SUMA DE VALORES")
numero = int(input("¿Cuántos valores va a introducir? "))

if numero <= 0:
    print("¡Imposible!")
else:
    suma = 0
    for i in range(1, numero + 1):
        valor = float(input(f"Escriba el número {i}: "))
        suma = suma + valor
    print(f"La suma de los números que ha escrito es {suma}")

Comentarios

  • El núcleo del programa utiliza un bucle para pedir al usuario el número de valores indicado por el usuario 
        for i in range(1, numero + 1):
        ...
  • Al pedir el número que se va a sumar, se utiliza la variable de control i: 
            valor = float(input(f"Escriba el número {i}: "))
  • El programa utiliza un acumulador ( suma) para calcular la suma de los números introducidos por el usuario.
  • Antes de pedir los valores el acumulador suma se pone a 0. 
        suma = 0
  • En cada iteración, el valor introducido por el usuario se añade al acumulador 
            suma = suma + valor
  • Una vez completado el bucle, se escribe el resultado. 
        print(f"La suma de los números que ha escrito es {suma}")

Bucle for (1) - C-2

print("SUMA ENTRE VALORES")
numero_1 = int(input("Escriba un número entero: "))
numero_2 = int(input(f"Escriba un número entero mayor que {numero_1}: "))

if numero_2 <= numero_1:
    print(f"¡Le he pedido un número entero mayor o igual que {numero_1}!")
else:
    suma = 0
    for i in range(numero_1, numero_2 + 1):
        suma = suma + i
    print(f"La suma desde {numero_1} hasta {numero_2} es {suma}")

    for i in range(numero_1, numero_2):
        print(f"{i} + ", end="")
    print(f"{numero_2} = {suma}")

Comentarios

  • El núcleo del programa utiliza un acumulador ( suma) para sumar los enteros entre los dos números introducidos por el usuario y escribe el resultado. Este código es muy similar al ejemplo comentado en la lección bucle for (1). 
        suma = 0
    for i in range(numero_1, numero_2 + 1):
        suma = suma + i
    print(f"La suma desde {numero_1} hasta {numero_2} es {suma}")
  • A continuación el programa tiene que escribir la expresión algebraica de la suma realizada (por ejejmplo, 10 + 11 + 12 + 13 + 14 = 60). Para ello, se realiza un bucle en el que en cada iteración se escribe la variable de control y el caracter +. Un detalle importante es que el bucle no debe llegar al último número (para que no escriba un + después del último número). 
        for i in range(numero_1, numero_2):
        print(f"{i} + ", end="")
  • Una vez completado el bucle, se escribe el último número, el carácter = y el resultado ya calculado antes. 
        print(f"{numero_2} = {suma}")

Bucle for (1) - C-3

print("MAYOR, MENOR Y MEDIA ARITMÉTICA")
numero = int(input("¿Cuántos valores va a introducir? "))

if numero <= 0:
    print("¡Imposible!")
else:
    valor = float(input("Escriba el número 1: "))
    minimo = maximo = suma = valor
    for i in range(2, numero + 1):
        valor = float(input(f"Escriba el número {i}: "))
        suma = suma + valor
        if valor < minimo:
            minimo = valor
        if valor > maximo:
            maximo = valor
    print(f"El número más pequeño de los introducidos es {minimo}")
    print(f"El número más grande de los introducidos es {maximo}")
    print(f"La media de los números introducidos es {suma / numero}")

Comentarios

  • La dificultad de este ejercicio consiste en que el programa debe detectar el número más pequeño y el más grande de los introducidos por el usuario. Para ello se utilizan dos variables auxiliares minimo y maximo.

    En cada iteración del bucle que va pidiendo valores al usuario, el programa comprueba si el valor introducido es menor que el mínimo o mayor que el máximo y, si es así, se guarda el nuevo valor.

            if valor < minimo:
            minimo = valor
        if valor > maximo:
            maximo = valor

    El problema es que las variables minimo y maximo necesitan tener un valor inicial, pero, independientemente del valor elegido, el usuario puede escribir números que no sean menores (o mayores) que el valor inicial y por tanto ese valor no se modificaría durante la ejecución del programa. Por ejemplo, si damos el valor 0 a mínimo y el usuario sólo escribe valores positivos, minimo no se modificaría.

    La solución es guardar como valor inicial de minimo y maximo el primer valor que introduzca el usuario.

        valor = float(input("Escriba el número 1: "))
    minimo = maximo = suma = valor
  • A continuación el programa pedirá el resto de valores al usuario. Como se ha pedido ya un valor, el bucle debe empezar en 2. 
        for i in range(2, numero + 1):
        ...
  • Al pedir el número que se va a sumar, se utiliza la variable de control i: 
            valor = float(input(f"Escriba el número {i}: "))
  • Para el cálculo de la media se necesita la suma de todos los valores introducidos, por lo que el programa incluye un acumulador suma que se modifica en cada iteración del bucle. 
            suma = suma + valor
  • El valor inicial del acumulador suma es también el primer valor escrito por el usuario. 
        minimo = maximo = suma = valor
  • Una vez completado el bucle, se escriben los resultados. La media es la suma de todos los valores dividida por el número de valores. 
        print(f"El número más pequeño de los introducidos es {minimo}")
    print(f"El número más grande de los introducidos es {maximo}")
    print(f"La media de los números introducidos es {suma / numero}")

Bucle for (1) - C-4

print("FACTORIAL")
numero = int(input("Escriba un número entero mayor que cero: "))

if numero <= 0:
    print("¡Le he pedido un número entero mayor que cero!")
else:
    factorial = 1
    for i in range(1, numero + 1):
        factorial = factorial * i
    print(f"El factorial de {numero} es {factorial}")

Comentarios

  • El núcleo del programa utiliza un acumulador ( factorial) para calcular el factorial del número indicado por el usuario. 
        factorial = 1
    for i in range(1, numero + 1):
        factorial = factorial * i
    print(f"El factorial de {numero} es {factorial}")
  • Como el factorial es el producto desde 1 hasta el número indicado por el usuario, range debe empezar en 1 y llegar hasta el número indicado por el usuario. 
        for i in range(1, numero + 1):
        ...
  • El acumulador factorial debe modificarse multiplicándose por la variable de control del bucle i. 
            factorial = factorial * i
  • El valor inicial del acumulador factorial debe ser 1, porque si valiera 0 al hacer multiplicaciones el resultado daría siempre cero.. 
        factorial = 1
  • Una vez completado el bucle, se escribe el resultado. 
        print(f"El factorial de {numero} es {factorial}")
