Bucle while - Ejercicios (1)

Bucle while (1)

Enunciados

  1. sí para continuar
  2. SI para terminar
  3. N para continuar
  4. no para terminar
  5. S o s para continuar
  6. S o s para terminar

Soluciones

  1. sí para continuar
  2. SI para terminar
  3. N para continuar
  4. no para terminar
  5. S o s para continuar
  6. S o s para terminar

Estos ejercicios corresponden a lo explicado en la lección sobre el bucle while.

Se pueden consultar unas posibles soluciones.
