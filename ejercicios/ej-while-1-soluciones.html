Bucle while - Soluciones (1)

Bucle while (1)

Enunciados

  1. sí para continuar
  2. SI para terminar
  3. N para continuar
  4. no para terminar
  5. S o s para continuar
  6. S o s para terminar

Soluciones

  1. sí para continuar
  2. SI para terminar
  3. N para continuar
  4. no para terminar
  5. S o s para continuar
  6. S o s para terminar

Se ofrecen a continuación unas posibles soluciones de los ejercicios (1) de bucle while.

Bucle while (1) - 1

MPTC

101

def main():
    print("DIGA sí PARA CONTINUAR")
    respuesta = input("¿Desea continuar el programa?: ")

    while respuesta == "sí":
        respuesta = input("¿Desea continuar el programa?: ")

    print("¡Hasta la vista!")


if __name__ == "__main__":
    main()
