Qué es mclibre Python testing

mclibre Python testing es una herramienta basa en PyTest que estoy desarrollando para ayudar a los alumnos a comprobar que han resuelto correctamente los ejercicios propuestos en este curso.

Actualmente (diciembre de 2018), mclibre Python testing no está preparada para ser utilizada, pero en esta lección quiero ir escribiendo las instrucciones de uso.

Instalar mclibre Python testing

mclibre Python testing requiere las bibliotecas PyTest y Requests.

Para interferir lo menos posible con la instalación de Python, se aconseja utilizar pipenv.

Los pasos para realizar la instalación son los siguientes:

  1. abra una ventana de terminal e instale pipenv: 
    pip install -U pipenv
  2. cree un directorio donde ejecutar mclibre Python testing
  3. abra una ventana de terminal en ese directorio
  4. cree un proyecto de pipenv en ese directorio: 
    pipenv install
  5. copie los ficheros de mclibre Python testing en ese directorio
  6. instale PyTest: 
    pipenv install pytest
  7. instale Requests: 
    pipenv install requests

Usar mclibre Python testing

Puede usar mclibre Python testing de dos maneras (XX es en ambos casos el número de ejercicio a comprobar):

  • directamente 
    pipenv run python mclibre-pytesting.py XX
  • o abriendo una subshell de pipenv 
    pipenv shell
python mclibre-pytesting.py XX

La ventaja de la subshell de pipenv es que no hay que escribir pipenv run cada vez que se quiere ejecutar el programa, pero el inconveniente es que es bastante limitada (por ejemplo, no se puede recuperar un comando anterior).

Para salir de la subshell de pìpenv, escriba

exit

Desinstalar mclibre Python testing

Los pasos para realizar la desinstalación son los siguientes:

  1. abra una ventana de terminal en el directorio en el que instaló mclibre Python testing
  2. elimine el entorno virtual: 
    pipenv --rm
  3. borre el directorio
